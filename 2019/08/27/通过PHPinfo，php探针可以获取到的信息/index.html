一句话木马的使用

前提的条件
1、需要知道远程目录
2、需要远程目录有写的权限
3、需要数据库开启了secure_file_priv(默认处于关闭状态，into outfile是无法写进去东西)

源代码 拷贝到网站目录中，之后会有什么操作
不修改 手动创建文件 root:root

第二种操作方式

mysql 日志写webshell

mysql每执行一个SQL语句，就会把SQL语句写入到文件日志中，但是默认情况下这个选项是关闭的。

因此，我们需要去开启这个选项，在设置日志路径，日志路径设置为网站目录，日志的文件名字为： .php 结尾的文件。

开启日志操作

必须有sql编辑框或者查询框才可以执行

SET GLOBAL general_log =’ON’

修改日志

SET GLOBAL general_log_file =’C:/phpstudy/绝对路径’
执行任意SQL语句，就能把内容存到.php文件中
