介绍

1）PHP是嵌入html页面中的脚本语言
2）目前最流行的网站开发语言
3）在几乎所有的平台中都可以运行
4）很多的企业都在使用PHP作为开发语言

转义

在PHP中除了单引号，任何字符放入单引号中，将没有任何意义
在PHP中双引号，放入特殊字符，将会有特殊的意义
EOF中变量有意义，其他特殊字符没有意义
转义： \ 普通字符串变为特殊字符串，特殊字符串变为普通字符
